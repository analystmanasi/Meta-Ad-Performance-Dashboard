# 📊 Meta Ad Performance Dashboard | Power BI Project

An interactive Power BI dashboard designed to analyze Meta advertising campaign performance across multiple social media platforms including Facebook, Instagram, and WhatsApp.

The dashboard provides insights into engagement metrics, conversion performance, audience behavior, campaign effectiveness, and overall advertising trends through dynamic visualizations and KPIs.

---

# 📌 Short Description / Purpose

The Meta Ad Performance Dashboard is a Power BI project created to monitor and evaluate digital advertising campaign performance using interactive analytics and visual storytelling.

This dashboard helps users analyze campaign engagement, clicks, impressions, conversion rates, audience demographics, and advertising effectiveness across different regions and ad types.

The project was built to practice real-world business analytics, dashboard development, KPI tracking, and data visualization using Power BI.

---

# 🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 Power BI Desktop – Main data visualization platform used to build the interactive report.
- 📂 Power Query – Used for data cleaning, transformation, and preprocessing.
- 🧠 DAX (Data Analysis Expressions) – Used for calculated measures, KPIs, and dynamic analysis.
- 🔗 Data Modeling – Relationships established between campaign and performance-related datasets.
- 📈 Data Visualization – Interactive charts, KPI cards, slicers, maps, and tables.
- 📁 File Format – `.pbix/.pbit` for dashboard development and `.png` for dashboard previews.

---

# 📂 Data Source

The dataset used for this project contains Meta advertising campaign performance data including:

- Campaign impressions
- Clicks and engagements
- Conversion metrics
- Purchase rates
- Audience demographics
- Geographic performance
- Ad types and campaign categories
- Budget and spending analysis

The dataset was used for practicing marketing analytics and business intelligence reporting.

# ✨ Features / Highlights

## 📌 Business Problem

Digital marketing campaigns generate large volumes of performance data daily, making it difficult to quickly identify:

- Which campaigns perform best
- Which audience groups are most engaged
- Which ad formats drive higher conversions
- Geographic regions with stronger engagement
- Budget efficiency and campaign ROI

Without proper dashboards, analyzing campaign performance becomes time-consuming and less effective.

---

## 🎯 Goal of the Dashboard

The primary goal of this dashboard is to:

- Transform raw advertising data into actionable insights
- Monitor campaign performance through interactive visuals
- Analyze engagement and conversion trends
- Compare ad performance across platforms and demographics
- Practice dashboard building and marketing analytics using Power BI

# 📊 Walkthrough of Key Visuals

## 🔹 KPI Cards

Displays key marketing performance indicators such as:

- Total Impressions
- Total Clicks
- Total Engagements
- Purchases
- Conversion Rate
- CTR (Click Through Rate)
- Total Budget
- Average Budget per Campaign

These KPIs provide a quick overview of advertising performance.

---

## 🔹 Engagements by Gender

A donut chart visualizing engagement distribution across:

- Male audience
- Female audience
- Other audience segments

Helps understand audience interaction patterns.

---

## 🔹 Engagements by Age

Histogram/bar chart displaying audience engagement trends by age groups.

Useful for identifying the most active customer demographics.

---

## 🔹 Weekly & Hourly Engagement Trends

Trend charts showing:

- Daily engagement activity
- Weekly performance patterns
- Hourly audience interaction trends

Helps identify peak engagement timings.

---

## 🔹 Geographic Analysis

Map visualization displaying engagement performance across different countries and regions.

Useful for regional campaign analysis and audience targeting.

---

## 🔹 Analysis by Ad Type

Performance comparison across ad formats such as:

- Image Ads

- Video Ads

- Impressions
- Clicks
- CTR

- Conversion Rate
